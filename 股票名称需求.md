在quant_trading_platform下创建一个scripts文件夹，在里面添加一个python文件，参考下面的示例代码，实现生成股票编码与股票名称之间的映射关系，供这个系统的其他功能模块以后使用。运行python用conda run -n qlib python xxx.py

参考代码：

import akshare as ak

import pandas as pd

# 获取映射

df = ak.stock_info_a_code_name()

df.to_csv("stock_code_name.csv", index=False)

# 后续读取

mapping = pd.read_csv("stock_code_name.csv").set_index("code")["name"].to_dict()

# 使用示例

print(mapping["SH600519"])  # 输出：贵州茅台
